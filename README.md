# ResearchAgentBench

ResearchAgentBench is a benchmark for evaluating research-oriented reasoning capabilities of frontier AI systems.

## Evaluated Models

- ChatGPT
- Claude
- DeepSeek
- Gemini

## Benchmark Categories

- Summarization
- Synthesis
- Gap Discovery
- Hypothesis Generation
- Research Planning
- Experiment Design

## Benchmark Statistics

- 15 Tasks
- 60 Evaluations
- 4 Frontier Models
- 5 Scoring Dimensions
